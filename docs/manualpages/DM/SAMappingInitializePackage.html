SAMappingInitializePackage

 This function initializes everything in the SAMapping package. It is called from PetscDLLibraryRegister() when using dynamic libraries, and on the first call to SAMappingCreate() when using static libraries.

Synopsis

#include "petscsa.h"  
PetscErrorCode  SAMappingInitializePackage(const char path[])

Input Parameter

path -The dynamic library path, or PETSC_NULL

Keywords

SAMapping, initialize, package

See Also

PetscInitialize()
