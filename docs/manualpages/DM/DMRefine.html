DMRefine

 Refines a DM object

Synopsis

#include "petscdm.h"     
PetscErrorCode  DMRefine(DM dm,MPI_Comm comm,DM *dmf)
Collective on DM

Input Parameter

dm - the DM object
comm - the communicator to contain the new DM object (or PETSC_NULL)

Output Parameter

dmf -the refined DM
