DMSlicedGetGlobalIndices

 Gets the global indices for all the local entries

Synopsis

#include "petscdmsliced.h"     
#include "petscmat.h"    
#include "petscmat.h"    
PetscErrorCode  DMSlicedGetGlobalIndices(DM dm,PetscInt *idx[])
Collective on DM

Input Parameter

slice -the slice object

Output Parameters

idx -the individual indices for each packed vector/array

Notes

The idx parameters should be freed by the calling routine with PetscFree()

See Also

DMSlicedDestroy(), DMSlicedCreateGlobalVector(), DMSlicedCreate()
